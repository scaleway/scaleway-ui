# Migrate SelectInput to SelectInputV2

Welcome to the migration of SelectInput into SelectInputV2, here we will see how to easily migrate to SelectInputV2 component.

Example:

```tsx
import { SelectInput } from '@ultraviolet/ui'

const MyComponent = () => (
  <SelectInput>
      My text
  </SelectInput />
)
```

Becomes:

```tsx
import { SelectInputV2 } from '@ultraviolet/ui'

const MyComponent = () => (
  <SelectInputV2>

)
```
