# `up()` `down()`

<Description>We provide two utilities to ease the media queries.</Description>

## Defined breakpoints

```
xsmall: 0
small: 576
medium: 768
large: 992
xlarge: 1200
```

## With a styled component

```js
import { down, up } from '../utilities'

const WithStyled = styled.div`
  background-color: red;
  height: 30px;
  line-height: 30px;
  width: 100%;
  text-align: center;

  ${down(
    'large',
    `
    background-color: blue;
  `,
  )}

  ${down(
    'medium',
    `
    background-color: green;
  `,
  )}

  ${down(
    'small',
    `
    background-color: yellow;
  `,
  )}
`
```

## With `css`

```js
const responsiveClass = css`
  background-color: red;
  height: 30px;
  line-height: 30px;
  width: 100%;
  text-align: center;

  ${up(
    'small',
    `
    background-color: blue;
  `,
  )}

  ${up(
    'medium',
    `
    background-color: green;
  `,
  )}

  ${up(
    'large',
    `
    background-color: yellow;
  `,
  )}
`

const WithCss = props => <div css={responsiveClass} {...props} />
```
