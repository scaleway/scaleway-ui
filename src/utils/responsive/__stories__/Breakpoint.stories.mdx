# Breakpoint

<Description>
  Breakpoint is a tiny React component to toggle the display of its chdilren at
  a specified breakpoint
</Description>

## Basic Usage

<Canvas>
  <Story name="With styled">
    <Breakpoint up="medium">
      Watch me disappear when screen is &lt; medium
    </Breakpoint>
    <Breakpoint down="medium">
      Watch me appear when screen is &gt; medium
    </Breakpoint>
  </Story>
</Canvas>
