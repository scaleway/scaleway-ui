## Custom component

<Description>
  The `creation` props indicates the starting time of the progression, by
  default it&apos;s the time of when the object is mounted.
</Description>
<Description>
  The `duration` props indicates an approximation of the progression duration in
  seconds, by default it&apos;s `120`
</Description>
<Description>
  You can chose the color of the Progression using the `color` props
</Description>
