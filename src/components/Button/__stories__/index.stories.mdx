## Progress

<Description>
  Add `ActivityIndicator` by setting `progress` property to `left` or `right`.
</Description>

<Canvas>
  <Story name="btn-progress">
    <div>
      {buttonSizes.map(size => (
        <Button key={`left-${size}`} size={size} progress="left" m={2}>
          {size} left progress
        </Button>
      ))}
    </div>
    <div>
      {buttonSizes.map(size => (
        <Button key={`right-${size}`} size={size} progress="right" m={2}>
          {size} right progress
        </Button>
      ))}
    </div>
    <div>
      {buttonSizes.map(size => (
        <Button key={size} size={size} progress m={2} />
      ))}
    </div>
  </Story>
</Canvas>

## Icons

<Description>
  Add icon using `icon`. You can specify the name of the icon or the icon
  itself.
</Description>

<Canvas>
  <Story name="btn-icons">
    {buttonSizes.map(size => (
      <React.Fragment key={size}>
        <Button size={size} icon="lock" m={2}>
          Icon + Text
        </Button>
        <Button size={size} icon="lock" m={2} />
      </React.Fragment>
    ))}
  </Story>
</Canvas>

## Icons Positions

<Description>
  Add a position icon using `iconPosition`. You can specify whether you want it
  to the left or to the right of your text. It requires an `icon`.
</Description>

<Canvas>
  <Story name="btn-icons-position">
    {['left', 'right'].map(position => (
      <Button key={position} iconPosition={position} icon="lock" m={2}>
        {position}
      </Button>
    ))}
  </Story>
</Canvas>

## Extend

<Description>
  Make button extensible using `extend` prop. It requires a `icon`.

<Canvas>
  <Story name="btn-extend">

        {size} extend
      </Button>
    ))}
  </Story>
</Canvas>
