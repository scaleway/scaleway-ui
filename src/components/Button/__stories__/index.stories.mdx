## Action

<Description>Action button is a icon only button with no padding</Description>

<Canvas>
  <Story name="Action">
    {iconsList.map(icon => (
      <Button action key={icon} icon={icon} mr={2} mb={2} />
    ))}
  </Story>
  <Story name="Action with variant">
    {buttonVariants.map(variant => (
      <Button action icon="lock" key={variant} variant={variant} mr={2} />
    ))}
  </Story>
  <Story name="Action rounded">
    {buttonVariants.map(variant => (
      <Button
        action="rounded"
        icon="lock"
        key={variant}

        mr={2}
      />
    ))}
  </Story>
</Canvas>

## Icons Positions

<Description>
  Add a position icon using `iconPosition`. You can specify whether you want it
  to the left or to the right of your text. It requires an `icon`.
</Description>

  <Story name="Icons Position">

## Link delegation

<Description>
  Using the `to` props, link behavior is delegated to [Link
  Component](docs-components-link).
</Description>

<Canvas>
  <Story name="Link delegation">
    <Button to="https://scaleway.com" target="_blank">
      Scaleway
    </Button>
  </Story>
</Canvas>

<Description>
  In case, you would like to disabled a link, this link will be transformed into
  a button.
</Description>

<Canvas>
  <Story name="Link delegation with disabled">
    <Button to="https://scaleway.com" target="_blank" disabled>
      Scaleway
    </Button>
  </Story>
</Canvas>
