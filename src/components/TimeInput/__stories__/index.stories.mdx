## TimeInput schedule

You can adjust the time between options with the `schedule` props.

`hours`, `half` and `quarter` are available.

By default, the `hours` option is selected.

<Canvas>
  <Story name="Schedule">
    <BlockList>
      <ControlValue value={{ value: '10:15', label: '10:15' }}>
        {({ value, onChange }) => (
          <TimeInput
            name="timeinput-test-6"
            placeholder="Time"
            value={value}
            onChange={onChange}
            schedule="quarter"
          />
        )}
      </ControlValue>
      <ControlValue value={{ value: '10:30', label: '10:30' }}>
        {({ value, onChange }) => (
