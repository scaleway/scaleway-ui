## Steps

You can change step size of your `Stepper` component. If you set it to 10 for example, your `Stepper` will increase
& decrease by steps of 10.

<Canvas>
  <Story name="Steps">
    {() => {
      const [value, setValue] = useState(0)
      return (
        <Stepper
          minValue={0}
          maxValue={100}
          step={10}
          onChange={setValue}
          value={value}
          width={200}
        />
      )
    }}
  </Story>
</Canvas>
