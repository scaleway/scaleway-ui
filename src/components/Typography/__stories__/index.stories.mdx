## Ellipsis

<Description>
  Force the text on a single line using `ellipsis` prop.
</Description>

<Canvas>
  <Story name="ellipsis">
    <strong>Without ellipsis</strong>
    <Box w={500} mt={1} mb={2}>
      <Typography>
        This text is quite long. Lorem ipsum dolor sit amet, consectetur
        adipisicing elit, sed do eiusmod tempor incididunt ut labore et dolore
        magna aliqua.
      </Typography>
    </Box>
    <strong>With ellipsis (a tooltip is displayed on hover)</strong>
    <Box w={500} mt={1} mb={2}>
      <Typography ellipsis>
        This text is quite long. Lorem ipsum dolor sit amet, consectetur
        adipisicing elit, sed do eiusmod tempor incididunt ut labore et dolore
        magna aliqua.
      </Typography>
    </Box>
  </Story>
</Canvas>
