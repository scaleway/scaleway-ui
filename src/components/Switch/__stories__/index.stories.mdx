# Switch

## Basic

<Description>
  `Switch` uses a `input` of type &quot;checkbox&quot; and they can only be used
  as controlled input.
</Description>

</Canvas>

## Labeled

Add the `labeled` prop you can choose where you want to put the label in the Switch. Possible positions are `left`, `right` or `inside` (it's the default label position when labeled is `true`). The width is automatically calculated. But you can override it by passing the `width` prop.

<Canvas>
  <Story name="Labeled">
    <BlockList>
      <ControlValue value>
        {({ value, onChange }) => (
          <Switch
            name="switch-label-right"
            labeled
            variant="primary"
            onLabel="CHECKED"
            offLabel="NOT CHECKED"
            checked={value}
            onChange={e => onChange(e.target.checked)}
          />
        )}
      </ControlValue>
      <ControlValue value>
        {({ value, onChange }) => (
          <Switch
            name="switch-label-right"
            labeled="inside"
            onLabel="CHECKED"
            offLabel="NOT CHECKED"
            width={140}
            variant="primary"
            checked={value}
            onChange={e => onChange(e.target.checked)}
          />
        )}
      </ControlValue>
      <div style={{ display: 'flex', justifyContent: 'space-between' }}>
        <ControlValue value>
          {({ value, onChange }) => (
            <Switch
              name="switch-label-left"
              labeled="left"
              onLabel="CHECKED"
              offLabel="NOT CHECKED"
              variant="primary"
              checked={value}
              onChange={e => onChange(e.target.checked)}
            />
          )}
        </ControlValue>
        <ControlValue value>
          {({ value, onChange }) => (
            <Switch
              name="switch-label-right"
              labeled="right"
              variant="primary"
              checked={value}
              onChange={e => onChange(e.target.checked)}
            />
          )}
        </ControlValue>
      </div>
    </BlockList>
  </Story>
</Canvas>

### Custom on/off texts

Default label placement is inside the Switch. you can customize the text with `onLabel` and `offLabel`props.

<Canvas>
  <Story name="Custom texts">
    <ControlValue value>
      {({ value, onChange }) => (
        <Switch
          name="switch-label-custom"
          labeled
          onLabel="YES"
          offLabel="NO"
          variant="primary"
          checked={value}
          onChange={e => onChange(e.target.checked)}
        />
      )}
    </ControlValue>
  </Story>
</Canvas>

### Custom Label render

If you choose to place label near the Switch component with `labeled` you can set a custom element for the label with the `onLabel` and `offLabel` props.

<Canvas>
  <Story name="Custom label render">
    <BlockList>
      <ControlValue value>
        {({ value, onChange }) => (
          <Switch
            name="switch-label-left"
            labeled
            onLabel={<Icon size={24} name="check-circle-outline" />}
            offLabel={<Icon size={24} name="close-circle-outline" />}
            variant="primary"
            checked={value}
            onChange={e => onChange(e.target.checked)}
          />
        )}
      </ControlValue>
      <ControlValue value>
        {({ value, onChange }) => (
          <Switch
            name="switch-label-left"
            labeled="left"
            onLabel={
              <Box
                width={100}
                display="flex"
                alignItems="center"
                justifyContent="flex-end"
              >
                <span>Enabled</span>&nbsp;
                <Icon size={24} name="check-circle-outline" />
              </Box>
            }
            offLabel={
              <Box
                width={100}
                display="flex"
                alignItems="center"
                justifyContent="flex-end"
              >
                <span>Disabled</span>&nbsp;
                <Icon size={24} name="close-circle-outline" />
              </Box>
            }
            variant="primary"
            checked={value}
            onChange={e => onChange(e.target.checked)}
          />
        )}
      </ControlValue>
    </BlockList>
  </Story>
</Canvas>
