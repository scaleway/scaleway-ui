# Radio

<Description>
  Radio only work as [controlled
  component](https://reactjs.org/docs/forms.html).
</Description>

## Controlled

<Canvas>
  <Story name="basic">
    <BlockList>
      <ControlValue value={false}>
        {({ value, onChange }) => (
          <>
            <Radio
              checked={value === 'choice-1'}
              value="choice-1"
              onChange={e => onChange(e.currentTarget.value)}
            >
              Choice 1
            </Radio>
            <Radio
              checked={value === 'choice-2'}
              value="choice-2"
              onChange={e => onChange(e.currentTarget.value)}
            >
              Choice 2
            </Radio>
          </>
        )}
      </ControlValue>
    </BlockList>
  </Story>
</Canvas>
