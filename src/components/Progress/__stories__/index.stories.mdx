## Clickable

<Description>
  By adding `onClick` prop on `Progress.Step` you can handle a click on a
  previous completed step.
</Description>

<Canvas>
  <Story name="Clickable">
    <ControlValue value={3}>
      {({ value, onChange }) => (
        <>
          <Progress selected={value}>
            <Progress.Step onClick={onChange} title="Step 1" />
            <Progress.Step onClick={onChange} title="Step 2" />
            <Progress.Step onClick={onChange} title="Step 3" />
          </Progress>
          <Button mt={1} onClick={() => onChange(3)}>
            Reset
          </Button>
        </>
      )}
    </ControlValue>
  </Story>
</Canvas>
