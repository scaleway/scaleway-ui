## With `userInputs` properties

### Description

`userInputs` properties can be used to specify which word shouldn't be used for a password.
That way you can force user to avoid using sensitive data such as: their email, login, name, etc.

In this example try to type `thisisalongpassword`, the score should be really low as the word has been "banned"
using `userInputs` properties.

<Canvas>
  <Story name="with userInputs">

      estimate={zxcvbn}
      strength={[

        { color: 'orange', t: 'weak' },
        { color: 'yellow', t: 'medium' },
        { color: 'green', t: 'strong' },
        { color: 'green', t: 'veryStrong' },
      ]}
      title="MyTitle"
      userInputs={['thisisalongpassword']}
    />
  </Story>
</Canvas>
