## Link Component

<Description>
  By default link is a `a`. You can specify a custom component (like React Router Link) by changing the theme entry `linkComponent`.
</Description>

## Absolute URLs

Absolute URLs (starting by `http` or `https`) are automatically detected in `to`. If detected, a
```jsx
<a href="..." />
```
is used instead of the default `linkComponent` specified in the theme.
