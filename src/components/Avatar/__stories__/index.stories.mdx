# Avatar

<Description>
  Show a small image that can be customized to be based on another image or a
  text.
</Description>

<Canvas>
  <Story name="Basic">
    <Avatar />
  </Story>
</Canvas>

## Image

<Description>
  You can change the default image by using the `image` prop. It work as `src`
  on a img tag.
</Description>

<Canvas>
  <Story name="Image">
    <Avatar image={support} />
  </Story>
</Canvas>

## Size

<Description>
  You can customize the size of the image by using the `size` prop.
</Description>

<Canvas>
  <Story name="Size">
    <Avatar size={48} />
  </Story>
</Canvas>

## Text

<Description>
  Instead of having an image you can put a text and it will take the best

</Description>

<Canvas>
  <Story
