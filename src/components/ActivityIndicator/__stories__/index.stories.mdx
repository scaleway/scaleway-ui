# ActivityIndicator

<Description>
  A component representing percentage as a circle which can be used to show
  activity
</Description>

## Basic

<Canvas>
  <Story name="Basic">
    <ActivityIndicator percentage={75} />
  </Story>
</Canvas>

## Percentage

<Description>
  You can set the percentage of completion with the `percentage` prop.
</Description>

<Canvas>
  <Story name="Percentage">
    {[8, 32, 50, 75, 90, 100].map(percentage => (
      <div style={{ display: 'inline-flex', marginRight: 8 }}>
        <ActivityIndicator
          key={`percent-${percentage}`}
          percentage={percentage}
        />
      </div>
    ))}
  </Story>
</Canvas>
