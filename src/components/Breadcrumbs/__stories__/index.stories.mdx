# Breadcrumbs

<Description>Creates a Breadcrumbs of hierarchical pages.</Description>
<Description>Link are automatically supported with `to` prop.</Description>

<Canvas>
  <Story name="Basic">
    <Breadcrumbs>
      <Breadcrumbs.Item to="/step1">Step 1</Breadcrumbs.Item>
      <Breadcrumbs.Item to="/step1/step2">Step 2</Breadcrumbs.Item>
      <Breadcrumbs.Item>Step 3</Breadcrumbs.Item>
    </Breadcrumbs>
  </Story>
</Canvas>

## Variants

## Active index

<Description>
  Active index is automatically determined as the last element. One can be
  specified using `activeIndex` prop.
</Description>

<Canvas>
  <Story name="Active Index">
    <Breadcrumbs activeIndex={1} variant="bubble">
      <Breadcrumbs.Item>Step 1</Breadcrumbs.Item>
      <Breadcrumbs.Item>Step 2</Breadcrumbs.Item>
      <Breadcrumbs.Item>Step 3</Breadcrumbs.Item>
    </Breadcrumbs>
  </Story>
</Canvas>
