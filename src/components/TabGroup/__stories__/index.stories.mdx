## Named

Every `Tab` can have a `name` prop set, like this the selected tab will not be the index of the tab but it&apos;s name

<Canvas>
  <Story name="Named">
    <ControlValue value="first">
      {({ value, onChange }) => (
        <TabGroup onChange={onChange} selected={value}>
          <TabGroup.Tab name="first">First</TabGroup.Tab>
          <TabGroup.Tab name="second">Second</TabGroup.Tab>
          <TabGroup.Tab name="third">Very long tab name</TabGroup.Tab>
        </TabGroup>
      )}
    </ControlValue>
  </Story>
</Canvas>

You can disable a `Tab` using the `disabled` prop

## Custom

You can change the component used for the `Tab` using the `as` prop

<Canvas>
  <Story name="Custom">
    <ControlValue value={1}>
      {({ value, onChange }) => (
        <TabGroup onChange={onChange} selected={value}>
          <TabGroup.Tab as="a">First</TabGroup.Tab>
          <TabGroup.Tab as="a">Second</TabGroup.Tab>
          <TabGroup.Tab disabled as="div">
            Disabled tab
          </TabGroup.Tab>
        </TabGroup>
      )}
    </ControlValue>
  </Story>
</Canvas>
