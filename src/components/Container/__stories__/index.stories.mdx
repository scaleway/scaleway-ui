# Container

<Description>
  Description wrap other component inside of it with title, sub-title, and other
  properties.
</Description>

<Canvas>
  <Story name="Basic">
    <Container>Hello</Container>
  </Story>
</Canvas>

## Title

<Canvas>
  <Story name="Title">
    <Container title="Hello there">General Kenobi</Container>
  </Story>
</Canvas>

## Right Title

<Canvas>
  <Story name="Right Title">
    <Container rightTitle="Hello there">General Kenobi</Container>
  </Story>
</Canvas>

## Sub Title

<Canvas>
  <Story name="Sub Title">
    <Container title="Hello there" subtitle="(To: General Grievous)">
      General Kenobi
    </Container>
  </Story>
</Canvas>

## Custom Header

<Description>
  Use `header` properties to add a component under title of your container. In
  this example we added a button.
</Description>

<Canvas>
  <Story name="Custom Header">
    <Container
      title="Hello there"
      header={<Button size="small" mb={1} icon="auto-fix" />}
    >
      General Kenobi
    </Container>
  </Story>
</Canvas>

## Disabled

<Description>
  Disabled properties will only disable children of the container. Title,
  sub-title and header will remain unchanged.
</Description>

<Canvas>
  <Story name="Disabled">
    <Container title="Hello there" disabled>
      General Kenobi
    </Container>
  </Story>
</Canvas>
