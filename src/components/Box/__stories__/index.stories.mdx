# Box

<Description>
  Box is base on [xstyled](https://xstyled.dev/docs/) `x.div`. You can use a
  `Box` for wrapping your components in it and apply style on them.
</Description>

<Description>
  In the example we have a `Box` with primary background, a height of `100px`,
  margin x `auto` and padding of `16px`.
</Description>

<Canvas>
  <Story name="Basic">
    <Box backgroundColor="primary" height={100} mx="auto" p={2} />
  </Story>
</Canvas>

## Bordered

<Description>
  Using prop `bordered` you can add a border and some padding on a Box.
</Description>

<Canvas>
  <Story name="Bordered">
    <Box bordered>This is a bordered box</Box>
  </Story>
</Canvas>

## Other element

<Description>
  Using prop `as` you can specify another html or React element instead of the
  default div.
</Description>

<Canvas>
  <Story name="Other element">
    <Box as="span">This is a Box as a span tag</Box>
  </Story>
</Canvas>
