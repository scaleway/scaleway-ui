# Welcome to Scaleway UI

Scaleway UI is a set of React components you can use to build your application. It is used to build our [Scaleway Console](https://console.scaleway.com).

## Getting started

### Installation

```bash
pnpm add @scaleway/ui
```

### Add a ThemeProvider

To use the library you need to put a `ThemeProvider` which come from `@emotion/react` library:

```jsx
import { ThemeProvider, styled } from '@emotion/react'
import { Text, theme } from '@scaleway/ui'

const CenteredText = styled(Text)`
  text-align: center;
`

export default function App() {
  return (
    <ThemeProvider theme={theme}>
      <CenteredText variant="headingLarge" as="h1">
        Welcome to Scaleway UI
      </CenteredText>
    </ThemeProvider>
  )
}
```

### Customizing

The default theme is available on Github, you can see it [here](https://github.com/scaleway/scaleway-ui/blob/main/src/theme/index.ts). For more customization go to [Colors page](/docs/theme-colors--palette)
