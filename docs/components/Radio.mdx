# Radio

## Controlled

Radio only work as [controlled component](https://reactjs.org/docs/forms.html).

<Playground>
  <BlockList>
    <ControlValue value={false}>
      {({ value, onChange }) => (
        <>
          <Radio checked={value === "choice-1"} value="choice-1" onChange={e => onChange(e.currentTarget.value)}>Choice 1</Radio>
          <Radio checked={value === "choice-2"} value="choice-2" onChange={e => onChange(e.currentTarget.value)}>Choice 2</Radio>
        </>
      )}
    </ControlValue>
  </BlockList>
</Playground>

## Size

Set size using `size` property.

<Playground>
  <BlockList>
    <Radio name="radio-size" size={10}>Small radio</Radio>
    <Radio name="radio-size" size={40}>Big radio</Radio>
  </BlockList>
</Playground>

## Disabled

<Playground>
  <BlockList>
    <Radio name="radio-size" disabled>Disabled radio</Radio>
  </BlockList>
</Playground>
